# 🏨 Nila Hotels Website

A modern and responsive hotel booking website built using **HTML, CSS, and JavaScript**. This project allows users to explore rooms, check availability, register/login, and pre-book hotel services online.

---

## ✨ Features

- 🏠 Attractive homepage with navigation
- 🏨 Room details (Deluxe Room, Executive Suite, Conference Hall)
- 📅 Check room availability by date
- 📝 Pre-booking system with form validation
- 🔐 User authentication (Register & Login)
- 💾 Local storage for user and booking data
- 🖼️ Image gallery section
- 📍 Google Maps location integration
- 💬 Simple chatbot assistant
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox, Animations, Responsive Design)  
- JavaScript (DOM Manipulation, Local Storage)
